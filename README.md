# tzcare-node

### To start off, follow this 3-step process.

1. Get the source code:

```
git clone https://github.com/tzcare/tzcare-node.git
cd tzcare-node
npm i
```

2. Rename  **.env_sample** to  **.env** then set the variables:

```
SERIAL_DEVICE=
SERIAL_SPEED=

NODE=
PUBLIC_KEY=
PRIVATE_KEY=
PUBLIC_KEY_HASH=
CONTRACT_ADDRESS=

AES_KEY=
IV=
```

3. Launch the Iot node:

```
node app.js
```
